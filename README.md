## 👨‍💻 How to Setup

```bash
  # Clone the project
  $ git clone https://github.com/Agotsilv/FortuneCookie
```
```bash
  # Enter directory
  $ cd FortuneCookie
```

```bash
  # Install the dependencies, if you use npm
  $ npm install
```
<p align="center">or</p>

```bash
  # Install the dependencies, if you use yarn
  $ yarn
```
